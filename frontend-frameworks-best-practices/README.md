# Frontend Frameworks Best Practices

Guidelines for writing clean, maintainable frontend code using React or Vue, combined with TailwindCSS.

This skill is intended for AI coding assistants to generate idiomatic frontend components that are performant and easy to read.

## Overview

Modern frontend development relies heavily on component-based architectures. The key to a maintainable codebase is ensuring components have a single responsibility, state is managed at the correct level, and CSS classes (especially utility classes like Tailwind) are organized to prevent visual clutter in the markup.

## Installing to Another Project

To install this specific skill into a target project using the GitHub CLI (`gh`), navigate to your target project's root directory and run:

```bash
gh skill install <OWNER>/<REPO> frontend-frameworks-best-practices
```

### Prerequisites
- GitHub CLI (`gh`) v2.90.0 or later installed and authenticated.
- Replace `<OWNER>/<REPO>` with the path of the repository hosting this skill library (e.g., `your-org/shared-skills`).
